## 📐 截图规格

| 要求项 | 规格 |
|--------|------|
| 分辨率 | 1920x1080（桌面端） |
| 格式 | PNG（无损） |
| 文件大小 | ≤ 2MB |
| 命名编码 | UTF-8 |

## 🔧 截图工具

- **Playwright**: `page.screenshot(path='...')`
- **Selenium**: `driver.save_screenshot('...')`
- **Chrome DevTools**: F12 → Cmd/Ctrl+Shift+P → Capture screenshot
