#### TC-MAT-001: 上传标准视频文件 [P0]

| 项目 | 内容 |
|------|------|
| **用例编号** | TC-MAT-001 |
| **用例名称** | 上传标准MP4视频文件 |
| **优先级** | P0 |
| **前置条件** | 服务已启动，网络正常 |
| **测试数据** | test_video_1080p.mp4 (1920x1080, 30s, ~15MB) |

**测试步骤**:

```http
POST /api/v1/materials/upload
Content-Type: multipart/form-data

file: test_video_1080p.mp4
type: video
name: 商品展示视频
```

**预期结果**:

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "fileId": "mat_20260323_xxx",
    "name": "商品展示视频.mp4",
    "type": "video",
    "url": "https://storage.example.com/materials/mat_20260323_xxx.mp4",
    "size": 15728640,
    "duration": 30.0,
    "resolution": "1920x1080",
    "thumbnail": "https://storage.example.com/thumbnails/mat_20260323_xxx.jpg",
    "createdAt": "2026-03-23T10:30:00Z"
  }
}
```

**验证点**:
- [ ] HTTP状态码为200
- [ ] 返回code为200
- [ ] fileId格式正确 (mat_YYYYMMDD_xxx)
- [ ] 文件已保存到存储
- [ ] 缩略图已生成
- [ ] 元数据已写入数据库

### B. 自动化测试脚本示例

```python
# tests/test_material_upload.py

import pytest
from fastapi.testclient import TestClient
from src.api.main import app

client = TestClient(app)

class TestMaterialUpload:
    """素材上传接口测试"""

    def test_upload_video_success(self):
        """测试上传标准视频文件"""
        with open("tests/fixtures/videos/test_video_1080p.mp4", "rb") as f:
            response = client.post(
                "/api/v1/materials/upload",
                files={"file": ("test_video.mp4", f, "video/mp4")},
                data={"type": "video", "name": "测试视频"}
            )

        assert response.status_code == 200
        data = response.json()
        assert data["code"] == 200
        assert "fileId" in data["data"]
        assert data["data"]["type"] == "video"

    def test_upload_unsupported_format(self):
        """测试上传不支持的文件格式"""
        with open("tests/fixtures/test_file.exe", "rb") as f:
            response = client.post(
                "/api/v1/materials/upload",
                files={"file": ("test_file.exe", f, "application/octet-stream")},
                data={"type": "video"}
            )

        assert response.status_code == 400
        data = response.json()
        assert data["code"] == 40001
```
