### 1.2 模块职责

| 模块 | 职责 | 关键接口 | 依赖 |
|------|------|----------|------|
| **素材管理服务** | 文件上传、存储、元数据管理 | upload, batch-upload | OSS, PostgreSQL |
| **模板服务** | 模板CRUD、参数验证、预览生成 | templates, templates/{id} | PostgreSQL, Redis |
| **任务服务** | 任务创建、状态管理、结果查询 | tasks, tasks/{id} | Celery, Redis |
| **特效服务** | 文字特效、关注特效、画中画渲染 | 内部调用 | MoviePy, FFmpeg |
| **渲染引擎** | 视频合成、特效应用、导出 | 内部调用 | MoviePy, FFmpeg |

### 1.3 数据流设计

```
用户操作 → API 网关 → API 服务 → 任务队列 → 渲染引擎 → 存储
                │                      │
                └────── 状态查询 ←─────┘
                        (Redis)
```

### 1.4 与阶段一系统集成

本项目基于阶段一已完成的 API 基础扩展：

| 阶段一模块 | 复用情况 | 本期扩展 |
|-----------|----------|----------|
| FileService | 复用 | 增加批量上传接口 |
| VideoService | 复用 | 增加特效渲染方法 |
| AudioService | 复用 | 无变化 |
| TaskService | 复用 | 增加批量任务支持 |
| **新增模块** | - | TemplateService, EffectService |

## 2. API 接口设计

### 2.1 接口概览

| 序号 | 接口名称 | HTTP方法 | 路径 | 描述 | 优先级 |
|------|----------|----------|------|------|--------|
| 1 | 上传素材 | POST | /api/v1/materials/upload | 上传单个视频/图片/文稿素材 | P0 |
| 2 | 批量上传 | POST | /api/v1/materials/batch-upload | 批量上传多个素材 | P0 |
| 3 | 获取模板列表 | GET | /api/v1/templates | 获取可用模板列表 | P0 |
| 4 | 获取模板详情 | GET | /api/v1/templates/{id} | 获取模板详细配置参数 | P0 |
| 5 | 创建生成任务 | POST | /api/v1/tasks | 创建视频生成任务 | P0 |
| 6 | 查询任务状态 | GET | /api/v1/tasks/{id} | 查询任务状态和结果 | P0 |
| 7 | 取消任务 | DELETE | /api/v1/tasks/{id} | 取消正在进行的任务 | P1 |
| 8 | 批量创建任务 | POST | /api/v1/tasks/batch | 批量创建多个生成任务 | P1 |
| 9 | 下载视频 | GET | /api/v1/download/{id} | 下载生成的视频 | P0 |
| 10 | 获取预览 | POST | /api/v1/preview | 生成预览视频 | P1 |

### 2.2 详细接口定义

---

#### 接口 1: 上传素材

**基本信息**

| 项目 | 内容 |
|------|------|
| 接口路径 | `POST /api/v1/materials/upload` |
| Content-Type | `multipart/form-data` |
| 功能描述 | 上传单个素材文件（视频/图片/文稿） |
| 优先级 | P0 |

**请求参数**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| file | binary | 是 | 素材文件 |
| type | string | 是 | 素材类型：`video` / `image` / `text` |
| name | string | 否 | 素材名称，默认取文件名 |
| description | string | 否 | 素材描述 |

**响应示例**

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "fileId": "mat_20260323_abc123",
    "name": "商品展示视频.mp4",
    "type": "video",
    "url": "https://storage.example.com/materials/mat_20260323_abc123.mp4",
    "size": 15728640,
    "duration": 30.5,
    "resolution": "1920x1080",
    "thumbnail": "https://storage.example.com/thumbnails/mat_20260323_abc123.jpg",
    "createdAt": "2026-03-23T10:30:00Z"
  }
}
```

**错误码**

| 错误码 | 含义 | 解决方案 |
|--------|------|----------|
| 40001 | 文件格式不支持 | 支持 mp4/mov/jpg/png/txt/md |
| 40002 | 文件大小超限 | 单文件最大 2GB |
| 40003 | 文件损坏 | 检查文件完整性 |

#### 接口 5: 创建生成任务

**基本信息**

| 项目 | 内容 |
|------|------|
| 接口路径 | `POST /api/v1/tasks` |
| Content-Type | `application/json` |
| 功能描述 | 创建视频生成任务 |
| 优先级 | P0 |

**请求参数**

```json
{
  "name": "双11促销视频",
  "sourceFiles": [
    { "fileId": "mat_20260323_abc123", "type": "video" },
    { "fileId": "mat_20260323_def456", "type": "image" }
  ],
  "templateId": "tpl_ecommerce_001",
  "parameters": {
    "textContent": {
      "main": "限时5折 仅限今日",
      "sub": "错过再等一年",
      "position": "bottom-center"
    },
    "textEffect": {
      "type": "flash",
      "style": {
        "font": "思源黑体 Bold",
        "size": 48,
        "color": "#FFD700",
        "stroke": { "color": "#000000", "width": 2 }
      }
    },
    "followEffect": {
      "enabled": true,
      "type": "pulse",
      "text": "关注店铺 领优惠券",
      "position": "bottom-right",
      "startTime": 25,
      "duration": 5
    },
    "pipEffect": { "enabled": false },
    "resolution": "1080p",
    "fps": 30
  },
  "callbackUrl": "https://example.com/webhook/video-complete"
}
```

```json
{

  "message": "success",
```

| 参数名 | 类型 | 必填 | 描述 |

**响应**

- Content-Type: `video/mp4`
- Content-Disposition: `attachment; filename="video_xxx.mp4"`
