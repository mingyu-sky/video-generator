## 📋 设计概述

### 项目背景
Video Generator 是一个 AI 驱动的视频批量生成平台，支持：
- 基础视频处理（拼接、特效、字幕、音乐）
- AI 短剧生成（剧本→分镜→视频→配音→字幕）
- 批量生产（多集短剧自动合成）

### 前端定位
- **目标用户**: 普通用户（非技术人员）
- **核心价值**: 降低使用门槛，可视化操作
- **技术栈**: React 18 + TypeScript + Ant Design 5
