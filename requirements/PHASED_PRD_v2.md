## 第一阶段：标准视频处理接口（12 人天）

### 功能清单（更新版）

| 编号 | 接口 | 功能描述 | 优先级 | 工作量 |
|------|------|----------|--------|--------|
| **基础设施** |
| 0.1 | POST /api/v1/auth/login | 用户登录 (JWT) | P0 | 1 人天 |
| 0.2 | POST /api/v1/upload | 视频/音频上传 | P0 | 2 人天 |
| 0.3 | GET /api/v1/tasks/:id | 任务进度查询 | P0 | 1 人天 |
| **核心功能** |
| 1.1 | POST /api/v1/video/concat | 视频拼接 | P0 | 2 人天 |
| 1.2 | POST /api/v1/video/text-overlay | 文字特效 | P0 | 1.5 人天 |
| 1.3 | POST /api/v1/video/image-overlay | 动图/图片水印 | P0 | 1.5 人天 |
| 1.4 | POST /api/v1/audio/voiceover | AI 配音生成 (Edge TTS) | P0 | 1.5 人天 |
| 1.5 | POST /api/v1/video/add-voiceover | 视频添加配音 | P0 | 1 人天 |
| 1.6 | POST /api/v1/video/add-music | 添加背景音乐 | P0 | 1 人天 |
| 1.7 | POST /api/v1/video/transition | 转场特效 | **P0** | 1.5 人天 |
| **配额管理** |
| 0.4 | GET /api/v1/quota | 查询配额 | P1 | 0.5 人天 |

### 技术指标

| 指标 | 目标 | 备注 |
|------|------|------|
| 视频处理速度 | ≥0.5x 实时 | 1080p 基准 |
| 并发能力 | 单用户 4 路/系统 20 路 | 队列管理 |
| 上传限制 | 单文件 ≤2GB | 支持断点续传 |
| API 响应 | ≤500ms (提交任务) | 不含渲染 |

### 技术架构

```
用户请求 → API 网关 (JWT 认证) → 任务队列 (Redis+Bull) → 工作节点 (FFmpeg)
                ↓                                              ↓
           返回 taskId                                    OSS 存储
                ↓
用户轮询 → GET /tasks/:id → 返回进度/结果
```
