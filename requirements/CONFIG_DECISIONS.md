## 技术方案影响

### 1. AI 配音 - Edge TTS
- **优势**: 免费、无需 API Key、支持多语言
- **依赖**: `edge-tts` Python 包
- **实现**: 集成到 `video_generator_v2.py`

### 2. ASR 字幕 - 阿里云
- **优势**: 准确率高、支持中文优化
- **依赖**: 阿里云智能语音 API
- **配置**: 需要 API Key（环境变量 `ALIYUN_ASR_KEY`）

### 3. 部署配置 - 4 核 8G
- **适用**: 第一期 MVP 验证
- **并发**: 支持单用户 4 路，系统 20 路（需队列管理）
- **成本**: 约 ¥200-300/月（阿里云 ECS）

### 4. 音乐库 - 免费
- **来源**: 
  - YouTube Audio Library
  - FreePD
  - Bensound（免费档）
- **版权**: 需标注来源，不可转售

### 5. 并发限制
- **单用户**: 4 个并发任务
- **系统**: 20 个并发任务（全局）
- **超额**: 进入队列等待
