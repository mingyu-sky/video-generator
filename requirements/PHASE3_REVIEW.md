### 3.2 视频处理页（5 人天）

#### 需求分析
| 功能模块 | 必要性 | 复杂度 | 备注 |
|----------|--------|--------|------|
| 文件上传 | P0 | 中 | 拖拽上传 + 进度条 |
| 参数配置 | P0 | 高 | 8 种处理类型，参数多 |
| 视频预览 | P0 | 高 | 需要播放器 + 时间轴 |
| 处理历史 | P1 | 低 | 保存用户配置模板 |

#### 接口依赖
```
POST /api/v1/files/upload        - 文件上传（已有）
GET /api/v1/files                - 文件列表（已有）
POST /api/v1/video/*             - 8 个视频处理接口（已有）
GET /api/v1/tasks/:id            - 任务进度（已有）
```

#### 审查意见
⚠️ **5 人天可能不足**，建议调整为 7 人天

**原因**:
1. 视频预览组件复杂度高（需要支持时间轴、多轨道预览）
2. 参数配置表单复杂（8 种处理类型，每种参数不同）
3. 需要处理大文件上传（分片上传 + 断点续传）

**建议补充**:
1. 添加"配置模板"功能，保存常用参数组合
2. 支持批量选择文件统一处理
3. 添加处理前后对比预览

**风险评估**: 中

---

### 3.3 AI 短剧页（5 人天）

#### 需求分析
| 功能模块 | 必要性 | 复杂度 | 备注 |
|----------|--------|--------|------|
| 主题输入 | P0 | 低 | 表单输入 |
| 剧本编辑 | P0 | 高 | 富文本编辑器 + JSON 预览 |
| 分镜预览 | P0 | 中 | 卡片式展示 |
| 生成控制 | P0 | 中 | 开始/暂停/取消 |
| 结果预览 | P0 | 高 | 多集视频列表 + 播放 |

#### 接口依赖
```
POST /api/v1/ai/script/generate     - 剧本生成（阶段二已实现）
GET /api/v1/ai/script/:id           - 获取剧本（阶段二已实现）
POST /api/v1/ai/storyboard/generate - 分镜生成（阶段二已实现）
POST /api/v1/ai/batch/generate      - 批量生成（阶段二已实现）
GET /api/v1/ai/batch/:id            - 查询进度（阶段二已实现）
```

#### 审查意见
⚠️ **5 人天严重不足**，建议调整为 8-10 人天

**原因**:
1. 剧本编辑器需要支持结构化编辑（场景/对话/情绪）
2. 分镜预览需要展示多个镜头，支持手动调整
3. 批量生成进度展示复杂（80 集 × 多镜头）
4. 需要处理 AI 生成失败的重试逻辑

**建议补充**:
1. 添加"剧本模板"功能，提供常用题材模板
2. 支持手动编辑分镜提示词（Prompt）
3. 添加"预览模式"，生成前预览关键帧
4. 支持部分重生成（单集/单镜头）

**风险评估**: 高
