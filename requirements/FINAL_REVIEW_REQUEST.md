**选项**：
- [ ] **Edge TTS** (推荐) - 免费，质量较好，支持多语言
- [ ] Azure TTS - 付费，质量最优，$0.01/1000 字符
- [ ] Google TTS - 付费，$0.006/1000 字符
- [ ] 暂不实现，仅支持上传配音文件

**建议**: 选择 Edge TTS (免费)，后续可切换到 Azure

#### 2. ASR 字幕识别服务

**选项**：
- [ ] **Whisper** (推荐) - 开源免费，准确率高，支持离线

- [ ] 腾讯云语音识别 - 付费，¥0.0025/秒
- [ ] 暂不实现，仅支持导入 SRT 文件

**建议**: 选择 Whisper (开源)，部署在本地

**建议**: 16 核/32GB/无 GPU/500GB SSD (初期)

**建议**: 初期使用免费音乐库
