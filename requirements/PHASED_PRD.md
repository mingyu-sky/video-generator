# Video Generator 四阶段发展规划

**版本**: v1.0  
**创建日期**: 2026-03-20  
**确认状态**: ⏳ 待老板确认

---

## 产品愿景

打造一站式 AI 视频生成平台，从基础视频处理到 AI 短剧创作，最终实现可视化操作。

---

## 第一阶段：标准视频处理接口

**目标**: 提供完整的视频后期处理 API 能力

### 功能清单

| 编号 | 接口 | 功能描述 | 优先级 |
|------|------|----------|--------|
| 1.1 | POST /api/v1/video/concat | 视频拼接（多视频合并） | P0 |
| 1.2 | POST /api/v1/video/text-overlay | 添加文字特效（位置/字体/动画） | P0 |
| 1.3 | POST /api/v1/video/image-overlay | 添加动图/图片水印 | P0 |
| 1.4 | POST /api/v1/audio/voiceover | AI 配音生成（Edge TTS） | P0 |
| 1.5 | POST /api/v1/video/add-voiceover | 视频添加配音 | P0 |
| 1.6 | POST /api/v1/video/add-music | 添加背景音乐 | P0 |
| 1.7 | POST /api/v1/video/transition | 添加转场特效 | P1 |

### 技术指标
- 处理速度：≥0.5x 实时
- 并发：单用户 4 路/系统 20 路
- 支持格式：MP4/MOV/AVI → MP4(H.264)

### 工作量估算
- 后端开发：5 人天
- 测试：2 人天
- **合计：7 人天**

---

## 第二阶段：AI 生成短剧能力

**目标**: 实现 AI 驱动的短剧自动化生产

### 功能清单

| 编号 | 模块 | 功能描述 | 优先级 |
|------|------|----------|--------|
| 2.1 | 剧本生成 | 基于主题/关键词生成剧本（GPT） | P0 |
| 2.2 | 分镜设计 | 根据剧本生成分镜描述 | P0 |
| 2.3 | AI 视频生成 | 调用第三方 API（https://cursorai.apifox.cn/） | P0 |
| 2.4 | 后期处理 | 调用第一阶段接口进行增强 | P0 |
| 2.5 | 批量生成 | 支持批量生成多集短剧 | P1 |

### 工作流程
```
主题输入 → 剧本生成 → 分镜设计 → AI 视频生成 → 后期处理 → 输出
```

### 工作量估算
- AI 集成：3 人天
- 流程编排：3 人天
- 测试：2 人天
- **合计：8 人天**
