## ⚠️ 踩坑记录

### 坑 1: API 参数设计不完整
**问题**: 音乐/配音接口缺少开始时间参数  
**原因**: 场景考虑不周全  
**解决**: 二次审查，补充 6 个参数  
**教训**: 参数设计需覆盖完整使用场景

### 坑 2: 测试用例遗漏
**问题**: 7 个接口无对应用例  
**原因**: 用例编写滞后于 API 设计  
**解决**: 建立 API-用例映射表  
**教训**: 用例与接口同步编写

### 坑 3: 第三方依赖未提前准备
**问题**: ASR 接口依赖阿里云 API，无法立即测试  
**原因**: 外部依赖未提前申请  
**解决**: Mock 数据先行，后续对接真实 API  
**教训**: 外部依赖需提前 1 周申请
